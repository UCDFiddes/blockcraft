# -help

The help command gives a great amount of help when using the command...

## Permissions

Members that are able to use the args command are:

* Anyone

The permissions that the bot needs are...

* Send Messages
* Embed Links
* Use External Emojis

## Format

{% hint style="info" %}
-help &lt;COMMAND&gt;
{% endhint %}
